argument vs parameter

发表于 2016-08-16

实参（argument/actual parameter）

是在调用时传递给函数的参数，可以是常量、变量、表达式、函数等，无论实参是何种类型的量，在进行函数调用时，它都必须有 确定的值，以便把这些值传递给形参。因此应预先用赋值，输入等方法使实参获得确定值。

形参（parameter）

它不是实际存在变量，所以又称虚拟变量。是在定义函数名和函数体的时候使用的参数，目的是用来接收调用改函数是传入的实参。在调用函数时，实参将赋值给形参。因而，必须注意实参的个数，类型应与形参对应。

CSS 选择器优先级

发表于 2016-08-13

如果同一个元素被不同的 css 选择器选中改变其同一个性质的样式，遵循哪个选择器的命令就要看哪个选择器的优先级高了，优先级是可以评分的，评分规则如下：

  1. 如果该性质是在 html 文档中的 <style> 元素中定义的，则加1000分
  2. 如果该性质后面加了 !important ,加100分，选择器中有 id 名称也加100分
  3. 选择器中有 class，pseudo-class，attribute，加10分
  4. 选择器中有 element，pseudo-element，加1分
    连在一起的都要算，如 p.nav:first-child 要算三个

一个可复制易维护的按钮

发表于 2016-08-13

设计一个按钮，

  1. 只需改变字体大小，其他的宽高，阴影大小，圆角半径等都会随之改变，而不用一个个修改
  2. 只需改变背景颜色，边框颜色，阴影颜色随之改变，不用一一修改
  • 要想其他性质随字体大小改变只需将其他性质的数值改为相应的 em 为单位的值，行高也可以设为单纯的数字，那也是字体大小的倍数
  • 边框，字体倒影，盒子倒影，都设为半透明的黑色，这样背景色改变时也可以搭
  • 渐变色设为半透明的白色渐变到完全透明

Flexbox

发表于 2016-08-12
  1. 父元素（flex container）中添加 display: flex; 使得子元素（flex items）以同样宽高的 column 排列
  2. flex-direction 可以指定排列的方向，有 row 和 column 可选
  3. 如果 row 排列撑不下可以叫 flex-wrap: wrap ，shorthand为 flex-flow: row wrap；
  4. 还可以指定各个栏的宽度，在子元素中加 flex：1 200px; 第一个数字要跟其他子元素对比才有意义，200px是设定了栏的最小尺寸
  1. align-items 是定义 flex item 垂直排列的，有 flex-start，flex-end，center也可以对子元素单独定义，align-self：flex-end/flex-start/center
  2. justify-content 是水平排列，有flex-start，flex-end，center，space-around，space-between
  1. 也可以在子元素中指定顺序，order：1；这里的1并不一定代表第一为，所有的子元素默认为0，1就要排到0后面，－1就排到0前面
